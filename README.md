Clothing Store Project

Website quản lý sản phẩm, khách hàng và đơn hàng cho cửa hàng quần áo (mô phỏng hệ thống bán hàng online).
Hệ thống hỗ trợ giỏ hàng, đặt hàng và quản lý tồn kho cơ bản.

Công nghệ sử dụng
- Java, Spring Boot  
- SQL Server  
- JPA, Thymeleaf  
- IntelliJ IDEA  

Vai trò
- Backend Developer: thiết kế entity, controller, service và kết nối cơ sở dữ liệu.
- Xử lý logic giỏ hàng và đơn hàng trong hệ thống.
